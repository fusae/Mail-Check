# 图表功能使用说明

## 概述

舆情报告系统现已支持自动生成4种可视化图表，让报告更加直观易读。

## 新增图表

### 1. 情感倾向饼图
**位置：** 第四章 - 情感分析与舆情态势
**内容：** 显示负面/中性/正面舆情的占比
**颜色：**
- 🔴 红色 - 负面
- 🟡 黄色 - 中性
- 🟢 绿色 - 正面

### 2. 问题类型饼图
**位置：** 第五章 - 舆情分类统计
**内容：** 显示各类问题的数量和占比（最多显示前6类）
**分类：** 医疗质量、服务态度、收费问题、环境卫生、流程问题、无障碍设施、隐私保护、其他

### 3. 舆情热度趋势折线图
**位置：** 第四章 - 情感分析与舆情态势
**内容：** 双Y轴折线图
- 左Y轴（蓝色）：舆情数量
- 右Y轴（红色）：平均风险分
**X轴：** 日期

### 4. 平台分布饼图
**位置：** 第二章 - 传播快照
**内容：** 显示各平台的舆情数量和占比

## 技术实现

### 图表生成流程

1. **数据准备** - 在 `generate_report_data()` 中生成报告数据
2. **图表生成** - 调用 `generate_charts()` 生成4个PNG图片
3. **图表嵌入** - 在Markdown/Word报告中插入图片引用

### 图表存储

- **存储位置：** `data/charts/`
- **命名规则：** `{图表类型}_{时间戳}.png`
- **分辨率：** 150 DPI
- **格式：** PNG

### 代码示例

```python
from report_generator_enhanced import EnhancedReportGenerator
import pandas as pd

# 创建报告生成器
generator = EnhancedReportGenerator()

# 准备数据
df = pd.read_csv('sentiment_data.csv')

# 生成报告数据（自动生成图表）
report_data = generator.generate_report_data(
    df=df,
    hospital_name='东莞市中医院'
)

# 查看生成的图表路径
chart_paths = report_data['chart_paths']
print(chart_paths)
# 输出：
# {
#     'sentiment_pie': '/path/to/sentiment_pie_20260304_140544.png',
#     'category_pie': '/path/to/category_pie_20260304_140544.png',
#     'trend_line': '/path/to/trend_line_20260304_140544.png',
#     'platform_pie': '/path/to/platform_pie_20260304_140544.png'
# }

# 生成Markdown报告（包含图表）
markdown_report = generator.generate_markdown_report(report_data)

# 生成Word报告（包含图表）
generator.generate_word_report(report_data, 'report.docx')
```

## 图表配置

### 中文字体设置

系统会自动尝试使用以下中文字体（按优先级）：
1. Arial Unicode MS（macOS）
2. SimHei（Windows）
3. DejaVu Sans（Linux）

如果中文显示异常，可以在代码中修改字体设置：

```python
# 在 report_generator_enhanced.py 的导入部分
plt.rcParams['font.sans-serif'] = ['你的字体名称']
```

### 图表样式自定义

可以修改以下参数来自定义图表样式：

**饼图：**
- `figsize` - 图表大小
- `colors` - 颜色列表
- `autopct` - 百分比格式
- `startangle` - 起始角度

**折线图：**
- `figsize` - 图表大小
- `linewidth` - 线条粗细
- `markersize` - 标记大小
- `dpi` - 分辨率

## 注意事项

1. **依赖库**
   - 需要安装 `matplotlib>=3.8.0`
   - 如果未安装，图表功能会自动跳过，不影响报告生成

2. **数据要求**
   - 至少需要2条以上的数据才能生成趋势图
   - 数据为空时会跳过相应图表

3. **文件管理**
   - 图表文件会保存在 `data/charts/` 目录
   - 建议定期清理旧的图表文件
   - 每次生成报告都会创建新的图表文件

4. **性能考虑**
   - 图表生成会增加约2-3秒的处理时间
   - 大数据量时建议使用采样

## 故障排查

### 问题1：图表不显示中文

**原因：** 系统缺少中文字体

**解决方案：**
```python
# 查看可用字体
import matplotlib.font_manager as fm
fonts = [f.name for f in fm.fontManager.ttflist]
print([f for f in fonts if 'Chinese' in f or 'Hei' in f])

# 设置找到的中文字体
plt.rcParams['font.sans-serif'] = ['找到的字体名称']
```

### 问题2：图表生成失败

**检查步骤：**
1. 确认 matplotlib 已安装：`pip list | grep matplotlib`
2. 检查 `data/charts/` 目录是否有写权限
3. 查看错误日志

### 问题3：Word中图表不显示

**原因：** 图片路径问题

**解决方案：**
- Markdown报告使用绝对路径，可以正常显示
- Word报告需要图片文件存在于指定路径
